# Farmacias Cruz Azul - Arquitectura Multicloud Segura
Este repositorio contiene la infraestructura, el código base y la documentación para el diseño seguro del sistema ERP de la cadena de **Farmacias Cruz Azul**, desarrollado para la asignatura **Arquitectura Multicloud**.
El proyecto implementa una arquitectura robusta basada en capas, contenedores y servicios gestionados (PaaS/IaaS), asegurando el control de acceso multifactor (MFA), aislamiento de red y automatización de respaldos en la nube de AWS bajo las mejores prácticas del *Well-Architected Framework*.
